=========================
silva.pageactions.mailpdf
=========================

This extension let the user to send the current page as a PDF in
`Silva`_. To generate the PDF it depends on silva.pageactions.pdf.

https://github.com/silvacms/silva.pageactions.mailpdf

Code repository
===============

You can find the code of this extension in Git:
https://github.com/silvacms/silva.pageactions.mailpdf
